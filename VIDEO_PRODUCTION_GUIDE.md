## **TECHNICAL RECORDING SETUP**

### **Screen Recording Settings:**
```
Resolution: 1920x1080 (1080p)
Frame Rate: 30fps
Audio: 48kHz, 16-bit minimum
Format: MP4 (H.264)
```

### **Browser Setup:**
```
- Use Chrome/Brave for best Web3 compatibility
- Clear all other tabs
- Disable notifications
- Use incognito mode for clean interface
- Zoom to 100% (no scaling)
```

### **MetaMask Setup:**
```
- Connect to Sepolia testnet
- Have test USDC and ETH ready
- Clear transaction history for clean demo
- Use a clean wallet address (not your main one)
```

3. **Audio Problems**: Record voiceover separately

- **Slow Loading**: Speed up video in post-production
- **UI Glitches**: Use smooth transitions to hide issues
- **Audio Sync**: Adjust timing in editing
- **Poor Quality**: Re-record critical sections only
