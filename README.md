### **Prerequisites**
```bash
Node.js 18+
npm or yarn
MetaMask wallet
Sepolia ETH (for gas)
```

### **1. Clone & Install**
```bash
git clone https://github.com/your-username/meta-army.git
cd Meta-Plot-AI
npm install
```

### **2. Environment Setup**
```bash
cp .env.example .env.local
# Add your API keys:
# - NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID
# - GEMINI_API_KEY
# - ETHERSCAN_API_KEY
```

### **3. Deploy Smart Contracts**
```bash
npm run compile
npm run deploy
# Copy contract addresses to .env.local
```

### **4. Start Development**
```bash
npm run dev
# Open http://localhost:8000
```

### **5. Connect Wallet & Test**
1. Connect MetaMask to Sepolia testnet
2. Get test tokens from faucets
3. Try: "Invest 10 USDC in Aave"
4. Approve the swarm and watch automation!
